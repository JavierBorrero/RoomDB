# Proyecto de prueba - Room

Este es un proyecto de prueba simple para aprender a utilizar Room.

La aplicación permite crear y eliminar contactos en una base de datos local. También es posible ordenar los contactos por nombre, apellido o número de teléfono.

No se utiliza inyección de dependencias (DI), ni se hace control de formularios, ya que el objetivo era crear un ejemplo sencillo y rápido para aprender a configurar Room.
